Mitten Parsing ToolKit (MPTK)

Complex Syntaxes Done Easily

The Mitten Parsing ToolKit was developed for the Mitten programming language, but it can be used for any compiler. It has a powerful and flexible parsing engine written in C++ that lets you create front-ends for even complex languages with only a few lines of code.

Source Code

include(std);

void main()
{
print("hello, world\n");
}

The small sample of Mitten source code above can be parsed with an equally small parser.

Lexical Tokenizer

Lexer lexer;

lexer.deliminate("\n") = Filtered;
lexer.deliminate(" ") = Filtered;
lexer.deliminate("\t") = Filtered;

lexer.deliminate("(");
lexer.deliminate(",");
lexer.deliminate(")");

lexer.deliminate("{");
lexer.deliminate(";");
lexer.deliminate("}");

lexer.deliminate("\"", "\"");

Semantic Parser

StructureParser parser;

parser.bind("(", ",", ")");
parser.bind("{", ";", "}");


























MPTK's brevity lets you quickly prototype fully-functional compilers while still keeping the integrity needed for industrial-strength front ends.

Efficient

MPTK's parsing engine only needs to run through the source code once to perform all the tasks it needs to generate easy-to-use ASTs. This means that it completes with other traditional parsing options.

Flexible

Every parser and tool contained in the MPTK engine can be extended with easy-to-use public callback members.

void myOnToken(Token t, vector<Token> &v)
{
cout << "'" << t.value() << "'\n";
}

lexer.onToken = myOnToken;


Not only are MPTK's tools extendable, but they are also highly configurable. The same lexer than is so effective for C-like programming languages is just as useful in handling markdown-like languages.
